# Python-Weather-App

This is a weather app I created. It uses WeatherAPI to track the weather of your location by Zip Code or Name.

## How To Use and Installation
To install, download the .py file from GitHub.

To use this program, you need to have a WetherAPI key, which is gained from here: https://www.weatherapi.com/

Slot that key into the "Key =" section of the code, and you are golden.

This app requires Python for use. Make sure it's installed.
